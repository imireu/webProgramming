Chapter 13: 비동기 프로그래밍과 웹 API

1. 비동기 처리와 fetch

fetch는 데이터를 서버에서 가져오는 비동기 함수입니다.

fetch('https://jsonplaceholder.typicode.com/posts')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

2. async/await

async function getPosts() {
  try {
    const res = await fetch('https://jsonplaceholder.typicode.com/posts');
    const data = await res.json();
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}
getPosts();

3. API란?

Application Programming Interface. 프로그램끼리 상호작용할 수 있도록 만든 인터페이스.

  • Open API: 누구나 사용할 수 있는 API (예: Kakao API, TMDB API)
  • REST API: HTTP 요청을 통해 자원을 CRUD 방식으로 처리

4. HTTP 요청 방식

  • GET: 데이터 조회
  • POST: 데이터 생성
  • PUT: 데이터 전체 수정
  • PATCH: 데이터 일부 수정
  • DELETE: 데이터 삭제
