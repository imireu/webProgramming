1. 내장 객체

자바스크립트에서 기본적으로 제공하는 객체로, 별도의 선언 없이 사용 가능합니다.

  • 대표 예: window, Math, Date, Array, String

2. window 객체

  • alert(), confirm(), prompt()

  • open(): 새 창 열기
window.alert("안녕하세요!");
window.open("https://example.com", "_blank", "width=400,height=300");

3. screen / history / location

  • screen.width, screen.height: 화면 크기
  • history.back(), history.forward()
  • location.href, location.reload()

4. Date 객체

시간 정보를 다루는 객체

const now = new Date();
const future = new Date("2024-12-25");

console.log(now.getFullYear(), now.getMonth(), now.getDate());
console.log(now.getHours(), now.getMinutes(), now.getSeconds());
console.log(now.getTime()); // 1970.1.1 이후 경과 시간(ms)

시간 단위 변환

  • 1초 = 1000ms
  • 1분 = 60초, 1시간 = 60분, 1일 = 24시간
  • 1일 = 1000 * 60 * 60 * 24 (ms 기준)

5. [실습] 만보 걷기 - 오늘까지 며칠째일까?

기록 시작일로부터 오늘까지 며칠이 지났는지를 계산합니다.

const firstDay = new Date("2023-01-01");
      const today = new Date();
      
      const passedTime = today.getTime() - firstDay.getTime();
      const passedDay = Math.floor(passedTime / (1000 * 60 * 60 * 24));
      
      document.querySelector("#result").textContent = passedDay + "일째입니다!";
