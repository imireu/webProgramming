Chapter 12: 캔버스 & 비동기 프로그래밍

1. Canvas 기본

<canvas> 태그는 HTML5에서 도입된 그래픽 요소로, 자바스크립트를 이용해 도형, 이미지, 애니메이션 등을 그릴 수 있는 요소입니다. 보통 게임이나 시각화 도구에 자주 활용됩니다.

<canvas id="myCanvas" width="300" height="150"></canvas>

위 코드는 id가 myCanvas인 300x150 크기의 캔버스를 생성합니다.

3. 비동기 프로그래밍

자바스크립트는 기본적으로 동기 처리 언어이지만, 서버 요청이나 타이머와 같은 작업은 비동기적으로 처리해야 효율적입니다. 비동기란, 코드가 순차적으로 실행되지 않고, 특정 작업이 끝나기를 기다리지 않고 다음 작업을 먼저 수행하는 방식입니다.

setTimeout 예시

setTimeout은 지정한 시간이 지난 후 특정 작업을 수행합니다.

console.log("1");
setTimeout(() => {
  console.log("2"); // 1초 후 실행됨
}, 1000);
console.log("3"); // 먼저 실행됨

출력 결과는 1 → 3 → (1초 후) 2 순서입니다.

콜백 함수

콜백 함수는 어떤 작업이 끝난 뒤 호출되는 함수입니다. 주로 비동기 작업 이후 실행될 로직을 정의할 때 사용합니다.

function sayHello(callback) {
  console.log("안녕");
  callback();
}
sayHello(() => {
  console.log("콜백 실행");
});

위 예시에서 "안녕"이 출력된 뒤 "콜백 실행"이 출력됩니다.
