Chapter 3: CSS(2) & JavaScript

CSS (2)

레이아웃이란?

웹 페이지 내 요소들을 배치하고 구조화하는 방식으로, 사용자 경험에 핵심적인 역할을 함.

CSS 레이아웃 설정 방식

  • 일반 흐름(Normal Flow): 요소가 기본 블록 또는 인라인 형식으로 위에서 아래로 쌓임
  • Positioning: static / relative / absolute / fixed / sticky
  • Flexbox: 일차원 레이아웃 구성. 주요 속성:
    • display: flex
    • flex-direction
    • justify-content (ex: center, space-between, space-around)
    • align-items (ex: stretch, center, flex-start, flex-end)
    • align-content
  • Grid: 2차원 레이아웃 구성. 주요 속성:
    • display: grid
    • grid-template-columns, grid-template-rows
    • grid-gap / gap
    • justify-items / align-items
    • place-items, place-content
  • Float: 요소를 좌우로 띄워 배치 (옛날 방식, clearfix 필요)
  • Table Layout: 테이블 기반 배치
  • Multi-column Layout: 신문/잡지처럼 다단 레이아웃 구성

Flexbox 예제

display: flex;
justify-content: center;
align-items: center;

Grid 예제

display: grid;
grid-template-columns: 1fr 2fr;
gap: 10px;

HTML5 레이아웃 태그 (시맨틱 태그)

  • header, nav, main, section, article, aside, footer

박스 모델

모든 HTML 요소는 박스 형태로 렌더링됨. 구성요소:

  • content → padding → border → margin

기타 CSS 레이아웃 관련 속성

  • display: block, inline, inline-block, none 등
  • visibility: visible, hidden
  • opacity: 투명도 조절 (0 ~ 1)

CSS 애니메이션

  • transition: 상태 변화 부드럽게 처리
  • transform: 2D/3D 변형 (rotate, scale, skew, translate)
  • animation: 키프레임 기반 애니메이션

애니메이션 예제

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

.box {
  animation: fadeIn 2s ease-in-out;
}

뷰포트와 반응형 디자인

  • 뷰포트: 사용자의 화면 영역
  • meta viewport 태그로 설정 가능

미디어 쿼리

기기 너비/높이, 해상도 등에 따라 스타일 적용:

@media (max-width: 768px) {
  body {
    background-color: lightgray;
  }
}

JavaScript

자바스크립트 정의

웹 페이지에 동적 기능을 추가하는 스크립트 언어로, 클라이언트 측에서 실행됨.

특징 및 배경

  • 인터프리터 방식
  • 동적 타입(dynamic typing)
  • 이벤트 기반(Event-driven)
  • 객체 기반(Object-based)

자바스크립트와 ECMAScript

ECMAScript는 자바스크립트의 표준 사양으로, JavaScript는 이를 구현한 언어.

ES6 (ECMAScript 2015)

  • let, const
  • 화살표 함수 (arrow functions)
  • 템플릿 리터럴
  • 클래스(class), 모듈(module)
  • Promise
  • 비구조화 할당(destructuring assignment)

클라이언트와 서버

자바스크립트는 주로 클라이언트에서 실행되지만, Node.js 등을 이용하면 서버에서도 실행 가능

자바스크립트 프레임워크 & 라이브러리

  • React, Vue.js, Angular, jQuery 등

프론트엔드 개발과 자바스크립트

HTML 구조 생성, CSS 조작, 사용자 입력 이벤트 처리 등 핵심 역할 수행

자바스크립트 작성 방식

  • 인라인 방식: 태그에 직접 작성
  • 내부 방식: <script> 태그 내부 작성
  • 외부 방식: .js 파일을 연결해 관리

작성 방식의 단점

인라인/내부 스크립트는 유지보수 어려움, 코드 중복, 가독성 저하 → 외부 스크립트 권장

DOCTYPE이란?

브라우저가 HTML 문서를 어떤 방식으로 해석해야 하는지 정의. 예: <!DOCTYPE html>

자바스크립트 예제

// 내부 스크립트 예시
<script>
  alert("Hello, world!");
</script>

// 외부 스크립트 연결
<script src="script.js"></script>
